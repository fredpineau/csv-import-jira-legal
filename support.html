How to use the app

  1. Open the app from Jira administration.
  2. Select a dropdown custom field.
  3. Select a field context.
  4. Upload a CSV file with one option per line.
  5. Click Preview to review changes.
  6. Click Apply import to update the options.

CSV format

Use one option value per line.

Known behavior

  • Missing values may be disabled.
  • Disabled values present in the CSV may be re-enabled.
  • You can optionally sort values alphabetically after import.
